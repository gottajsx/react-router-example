## Install Project

```
npx create-react-app@latest router-example
```

## Install Dependencies

```
npm i react-router-dom
```

## Start the App

Runs the app in the development mode.
```
npm run start
```
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
